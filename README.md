# Superstore Retail Sales Analysis

## Project Overview
This project focuses on analyzing Superstore retail sales data to identify business trends, sales performance, profitability, and customer insights using **MySQL** and **Power BI**.

The objective of this project is to transform raw retail data into actionable business insights through SQL analysis and interactive dashboards.

---

## Business Objectives
- Analyze overall sales performance
- Identify top-performing categories and sub-categories
- Track profit trends across regions
- Understand customer purchasing behavior
- Create an interactive dashboard for decision-making

---

## Tools & Technologies Used
- **MySQL** – Data Cleaning & Analysis
- **Power BI** – Dashboard Development & Visualization
- **Excel/CSV** – Dataset Handling
